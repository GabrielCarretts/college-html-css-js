# Web Development Fundamentals

## Description

This repository contains a collection of projects developed while learning the fundamentals of web development. The focus is on building a solid understanding of HTML structure, CSS styling, and basic user interface design.

Each project explores different concepts such as forms, layouts, and responsive design, with an emphasis on clean code and progressive improvement.

---

## Projects

### 1. User Registration Form

A structured form that collects personal data and user preferences.

**Concepts:**

* HTML forms and input types
* Labels and accessibility
* Field grouping with `fieldset` and `legend`
* Basic validation (`required`)
* CSS styling for layout and usability

---

### 2. News Portal Page

A simple news-style webpage layout with navigation and article content.

**Concepts:**

* Semantic HTML (`header`, `nav`, `section`, `article`)
* Navigation menus
* Content organization
* Basic page layout with CSS

---

### 3. JUNIPLAC Landing Page

A responsive landing page for a university sports event.

**Concepts:**

* Component-based layout (cards, containers)
* CSS Grid and Flexbox
* Responsive design
* UI/UX improvements (spacing, hierarchy, buttons)
* CSS variables and reusable styles

---

## Technologies Used

* HTML5
* CSS3 (Flexbox, Grid, responsive design)

---

## What I Learned

* Structuring web pages using semantic HTML
* Creating and organizing forms
* Styling layouts with CSS
* Improving UI/UX through spacing, alignment, and hierarchy
* Building responsive and reusable components
