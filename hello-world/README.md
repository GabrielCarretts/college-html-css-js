# User Registration Form

## Description
A user registration form built with HTML that collects personal information and user preferences. The form is structured using semantic elements and grouped inputs to improve organization and usability.

## Features
- Personal data fields (name, email, password, date of birth)
- Additional information (gender, interests, country)
- File upload (profile picture)
- Color picker (favorite color)
- Satisfaction level slider (0–10)
- Text area for user biography
- Form validation using required fields
- Submit and reset functionality

## Technologies
- HTML5
- Form Elements (input, select, textarea)
- Semantic Structure (form, fieldset, legend, label)

## Concepts Applied
- Form structure and data collection
- Input types and validation
- Grouping fields with fieldset and legend
- Accessibility using labels
- Handling multiple selections (checkboxes)

## How to Run
1. Download or clone the repository
2. Open the `index.html` file in your browser
