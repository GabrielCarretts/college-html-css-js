# User Registration Form

## Description
A structured user registration form built with HTML and styled with CSS. The form collects personal data, preferences, and additional user information, using semantic elements to ensure clarity and accessibility.

## Features
- Personal data collection (name, city, email, password, date of birth)
- Gender selection using radio buttons
- Multiple interests selection using checkboxes
- Country selection via dropdown menu
- Profile picture upload
- Color picker for favorite color
- Satisfaction level slider (0–10 scale)
- User biography text area
- Form validation with required fields
- Submit and reset functionality

## Technologies
- HTML5
- CSS3 (Flexbox, layout styling)
- Form Elements (input, select, textarea)

## Concepts Applied
- Semantic form structure (`form`, `fieldset`, `legend`)
- Accessibility with properly linked `label` and `input`
- Input validation using HTML attributes
- Grouping related data fields
- Handling multiple values (checkbox arrays)
- Responsive and centered layout design

## Styling Highlights
- Clean card-style form layout
- Organized spacing and alignment using Flexbox
- Custom styling for inputs, buttons, and range slider
- Visual hierarchy using typography and spacing

## How to Run
1. Download or clone the repository
2. Open the `index.html` file in your browser
