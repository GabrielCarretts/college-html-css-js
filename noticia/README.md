# News Portal Page

## Description
A simple news-style webpage built with HTML and CSS. The project simulates a basic news portal layout, featuring a header, navigation menu, and an article section with text and image content.

## Features
- Header with page title
- Navigation menu with basic links
- Article section displaying a news post
- Image integration with descriptive alt text
- Clean and centered content layout

## Technologies
- HTML5
- CSS3 (Flexbox, layout styling)

## Concepts Applied
- Semantic HTML structure (`header`, `nav`, `section`, `article`)
- Page layout and content organization
- Navigation menu styling with Flexbox
- Basic responsive design principles
- Image handling and accessibility (`alt` text)

## Styling Highlights
- Dark header and navigation bar for contrast
- Centered content container
- Card-style article with shadow and rounded corners
- Hover effects on navigation items

## How to Run
1. Download or clone the repository
2. Open the `index.html` file in your browser
