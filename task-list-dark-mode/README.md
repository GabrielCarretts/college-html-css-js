# Task List App

## Description
A simple task list application built with HTML, CSS, and JavaScript. The project allows users to manage tasks by adding, completing, and removing items, while also supporting light and dark theme switching.

## Features
- Add new tasks dynamically
- Mark tasks as completed or pending
- Delete tasks from the list
- Toggle between light and dark themes
- Live task counters (total, completed, pending)

## Technologies
- HTML5
- CSS3 (variables, layout, responsive design)
- JavaScript (DOM manipulation, event handling)

## Concepts Applied
- DOM manipulation
- Event listeners and event delegation
- Form handling with `preventDefault()`
- Dynamic element creation
- UI state updates (task counters)
- Theme switching using CSS classes

## How to Run
1. Download or clone the repository
2. Open the `index.html` file in your browser
